# Экран локализации (LocalizationPage)

## Описание
Страница тестирования различных сценариев ошибок API.

## Структура экрана

### Список сценариев

│  с телом { "login": "...", "password": "..." }.    │
│                                                     │
│  Сервер должен ответить статусом 200 OK и вернуть  │

│  │ [1] Сценарий 1                                │ │
