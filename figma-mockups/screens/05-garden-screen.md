## Состояния грядки

### Пустая грядка
- **Условие:** !plot.type || plot.harvested
- **Отображение:** 
  - Текст: "Пустая грядка #{slot}"
  - Кнопка: "Посадить"

### Растёт (не созрело)
- **Условие:** plot.type && !plot.matured
- **Отображение:**
  - Иконка + название культуры
  - Таймер обратного отсчёта
  - Кнопка "Выкопать" (если canUproot)

### Созрело
- **Условие:** plot.matured
- **Отображение:**
  - Иконка + название культуры
  - Кнопка "Собрать"
  - Кнопка "Выкопать" (если canUproot)

## Seed Picker

### Открытие
- Клик на "Посадить" открывает список семян

### Выбор
- Клик на "Выбрать" сажает семя и закрывает модалку

### Пустой инвентарь
- **Текст:** "Нет семян"

## Таймер созревания

### Формат
- **MM:SS** (минуты:секунды)
- Пример: `5:30`, `0:45`, `10:00`

### Логика
```
total = growthMinutes * 60 * 1000
passed = now - plantedAt
left = max(0, total - passed)
minutes = floor(left / 60000)
seconds = floor((left % 60000) / 1000)
```

## Данные

```typescript
interface Plot {
  slot: number;
  type: SeedType | null;
  plantedAt: string | null;
  matured: boolean;
  harvested: boolean;
}

interface GardenResponse {
  plots: Plot[];
  growthMinutes: number;
  canUproot: boolean;
}

interface InventorySeed {
  id: number;
  type: SeedType;
}
```
