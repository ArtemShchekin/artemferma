## Состояния

### Активный контрол
- При клике на ингредиент появляется слайдер
- Только один слайдер активен одновременно

### Недостаточно ингредиентов
- **Сообщение:** "Недостаточно ингредиентов для [фруктового/овощного] салата"
- **Цвет:** красный (#e84c3d)

### Неправильное количество
- **Сообщение:** "Выстави ползунки на нужное количество"
- **Цвет:** красный (#e84c3d)

### Успешное приготовление
- **Toast:** "Салат приготовлен и съеден!"
- Или: "Салат приготовлен, но не съеден"

## Логика валидации

### Проверка достаточности
```typescript
hasEnoughIngredients(state, recipe) {
  for (type, amount in recipe.veg) {
    if (state.vegetables[type] < amount) return false;
  }
  if (recipe.liquids.yogurtMl && state.yogurtMl < recipe.liquids.yogurtMl) return false;
  if (recipe.liquids.sunflowerOilMl && state.sunflowerOilMl < recipe.liquids.sunflowerOilMl) return false;
  return true;
}
```

### Проверка точности
```typescript
isSelectionExact(selection, recipe) {
  for (type, amount in recipe.veg) {
    if (selection[type] !== amount) return false;
  }
  if (recipe.liquids.yogurtMl && selection.yogurtMl !== recipe.liquids.yogurtMl) return false;
  if (recipe.liquids.sunflowerOilMl && selection.sunflowerOilMl !== recipe.liquids.sunflowerOilMl) return false;
  return true;
}
```
