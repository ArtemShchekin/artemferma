## Компоненты

### Mode Tabs
```
[🛒 Покупка]  [💰 Продажа]
   ← active
```
- **Display:** flex, gap: 8px
- **Стиль:** как обычные табы
- **Иконки:** btn-buy.svg, btn-sell.svg

### Shop Item (Buy Mode)
```
┌─────────────────────────────────────────┐
│ [icon] Название           [Купить]      │
└─────────────────────────────────────────┘
```
- **Class:** shop-item
- **Display:** flex, justify-content: space-between, align-items: center
- **Border:** 2px dashed #7a4f2c
- **Border-radius:** 12px
- **Padding:** 8px 12px
- **Background:** #fff

#### Shop Left
- **Display:** flex, align-items: center, gap: 10px
- **Icon:** 36×36px

### Buy Button
- **Стиль:** btn primary
- **Иконка:** btn-buy.svg
- **Текст:** "Купить" или "Купить за X ₽"

### Sell Button
- **Стиль:** btn primary
- **Иконка:** btn-sell.svg
- **Текст:** "Продать"

## Секции покупки

### Семена уровня 1 (BASE_SEED_TYPES)
| Тип | Иконка | Название |
|-----|--------|----------|
| carrot | carrot.svg | Морковь |
| cabbage | cabbage.svg | Капуста |
| radish | radish.svg | Редис |

### Семена уровня 2 (ADVANCED_SEED_TYPES)
| Тип | Иконка | Название | Требование |
|-----|--------|----------|------------|
| potato | potato.svg | Картофель | Уровень 2 |
| eggplant | eggplant.svg | Баклажан | Уровень 2 |
| mango | mango.svg | Манго | Уровень 2 |

- **Opacity:** 0.5 если level < 2
- **Button disabled:** если level < 2

## Секция продажи

### Пустой инвентарь
- **Текст:** "Пусто"

### Товары на продажу
- Только vegWashed (мытые овощи)
- Отображаются с иконкой и названием
- Кнопка "Продать"

## Данные

```typescript
interface PricesResponse {
  purchase: { basePrice: number; advPrice: number };
  sale: { basePrice: number; advPrice: number };
  supplies: { 
    yogurt: { price: number; volume: number }; 
    sunflowerOil: { price: number; volume: number } 
  };
}

interface InventoryItem {
  id: number;
  type: SeedType;
}

interface InventoryResponse {
  seeds: InventoryItem[];
  vegRaw: InventoryItem[];
  vegWashed: InventoryItem[];
}
```
