# Экран профиля (ProfileTab)

## Описание
Страница профиля фермера с редактированием данных, статистикой и информацией о запасах.

## Структура экрана

```
┌─────────────────────────────────────────────────────┐
│  Badges Row                                         │
│  [Ур.5] [Продано:10] [Баланс:100₽] [Салатов:2]...  │
├─────────────────────────────────────────────────────┤
│  Profile Card                                       │
│  ┌───────────────────────────────────────────────┐ │
│  │ ☐ Ты крутой фермер?                           │ │
│  │                                               │ │
│  │  Обычный режим (не cool farmer):              │ │
│  │  ┌──────────┐  ┌──────────┐                  │ │
│  │  │ Имя      │  │ Фамилия  │                  │ │
│  │  │ [____]   │  │ [____]   │                  │ │
│  │  └──────────┘  └──────────┘                  │ │
│  │  ┌──────────┐                                │ │
│  │  │ Отчество │                                │ │
│  │  │ [____]   │                                │ │
│  │  └──────────┘                                │ │
│  │                                               │ │
│  │  Cool Farmer режим:                           │ │

│  └───────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────────┤
│  Prices & Stock Card                               │
│  ┌───────────────────────────────────────────────┐ │
│  │ Цены и запасы                                 │ │
│  │                                               │ │
│  │ [Йогурта: 500 мл] [Масла: 200 мл]            │ │
```

## Компоненты

### Badges Row
```
[Уровень: 5] [Продано: 10] [Баланс: 100 ₽] [Салатов съедено: 2] [Ты худой] [Фермер: Иван]
```
- **Display:** flex, gap: 8px, justify-content: flex-end
- **Badge стиль:**
  - Фон: #fff
  - Граница: 2px solid #7a4f2c
  - Border-radius: 999px
  - Padding: 4px 10px
  - Font: 14px, bold, #7a4f2c

#### Money Badge
- **Класс:** money
- **Цвет:** #1b5e20
- **Font-weight:** 800

### Profile Card

#### Checkbox
- **Label:** "Ты крутой фермер?"
- **Gap:** 8px между checkbox и текстом

#### Input Fields (Grid 2 columns)
- **Display:** grid, grid-template-columns: repeat(2, 1fr)
- **Gap:** 12px
- **Margin-top:** 12px

##### Обычный режим поля:
- Имя
- Фамилия  
- Отчество

##### Cool Farmer режим поля:
- Никнейм (валидация: `^[A-Za-z]{2,15}$`)
- Паспорт (валидация: `^\d{6}$`)

#### Валидация полей
- **Русские имена:** `^[А-ЯЁа-яё\-\s]{2,40}$`
- **Никнейм:** `^[A-Za-z]{2,15}$`
- **Паспорт:** `^\d{6}$` (ровно 6 цифр)

#### Save Button
- **Align:** flex-end
- **Margin-top:** 16px
- **Текст:** "Сохранить"

### Prices & Stock Card

#### Заголовок
- **H3:** "Цены и запасы"
- **Margin-top:** 12px

#### Бейджи запасов
```
[Йогурта на складе: 500 мл]
[Масла на складе: 200 мл]
[Морковь на складе: 10 шт.]
[Капуста на складе: 5 шт.]
[Редис на складе: 3 шт.]
[Картофель на складе: 8 шт.]
[Баклажан на складе: 2 шт.]
[Манго на складе: 4 шт.]
```

### Fitness Status Badge
- **Условие:** saladsEaten >= 3 ? "Ты толстый" : "Ты худой"

## Состояния

### Загрузка
- **Отображение:** "Загрузка..." или сообщение об ошибке

### Ошибка загрузки
- **Текст:** "Не удалось загрузить профиль"

### Обычный фермер
- Поля: firstName, lastName, middleName
- Валидация на кириллицу

### Крутой фермер (cool farmer)
- Поля: nickname, passport
- Валидация на латиницу и цифры

## Данные профиля

```typescript
interface ProfileResponse {
  isCoolFarmer: boolean;
  firstName: string | null;
  lastName: string | null;
  middleName: string | null;
  nickname: string | null;
  passport: string | null;
  level: number;
  soldCount: number;
  balance: number;
  yogurtMl: number;
  sunflowerOilMl: number;
  saladsEaten: number;
  isFatFarmer: boolean;
  isAdmin: boolean;
}
```
