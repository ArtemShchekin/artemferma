## Описание
Страница управления предметами: семена, свежие овощи, мытые овощи и протухшие овощи.

## Секции инвентаря

### 1. Семена (seeds)
- **Отображение:** иконка + название + дата получения
- **Действия:** нет (только просмотр)
- **Дата формат:** `new Date(created_at).toLocaleString()`

### 2. Свежесобранные овощи (vegRaw)
- **Отображение:** иконка + название
- **Действие:** кнопка "Помыть"
- **API:** PATCH /inventory/wash/{id}

### 3. Готовые к продаже овощи (vegWashed)
- **Отображение:** иконка + название + статус
- **Действия:** нет (можно продать в магазине)

### 4. Протухшие овощи (vegRotten)
- **Отображение:** иконка + название
- **Действие:** кнопка "Выкинуть"
- **API:** DELETE /inventory/{id}

## Состояния

### Пустая секция
- **Текст:** "Пусто"
- **Отображение:** вместо списка предметов

### Загрузка
- Показываются текущие данные
- Обновление после действий

## Данные

```typescript
interface InventoryItem {
  id: number;
  type: SeedType;
  status: string;
  created_at: string;
  is_rotten?: boolean;
}

interface InventoryResponse {
  seeds: InventoryItem[];
  vegRaw: InventoryItem[];
  vegWashed: InventoryItem[];
  vegRotten: InventoryItem[];
}
```

## Действия

### Помыть овощ
- **Endpoint:** PATCH /inventory/wash/{inventoryId}
- **Результат:** vegRaw → vegWashed
- **Toast:** "Вымыто"

### Выкинуть протухший
- **Endpoint:** DELETE /inventory/{inventoryId}
- **Результат:** удаление из vegRotten
- **Toast:** "Выброшено"
