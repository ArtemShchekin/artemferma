## 🔧 Как использовать в Figma

1. **Импорт SVG:**
   - Перетащите файлы `.svg` прямо в Figma
   - Все элементы будут импортированы как векторные объекты

2. **Создание компонентов:**
   - Выделите повторяющиеся элементы (кнопки, карточки, табы)
   - Нажмите `Ctrl+Alt+K` (Win) или `Cmd+Option+K` (Mac) для создания компонента

3. **Настройка стилей:**
   - Создайте Color Styles для цветовой палитры
   - Создайте Text Styles для типографики
   - Создайте Effect Styles для теней

4. **Auto Layout:**
   - Примените Auto Layout к карточкам и спискам
   - Настройте padding и gap согласно спецификации
