# Типографика

## Шрифт

**Основной шрифт:** FarmHand (Comic Sans MS как fallback)

```css
font-family: 'FarmHand', 'Comic Sans MS', cursive, system-ui;
```

## Размер и начертание

| Элемент | Размер | Вес | Line-height | Цвет |
|---------|--------|-----|-------------|------|
| H1 | 28px | 700 | 1.2 | #7a4f2c |
| H2 | 24px | 700 | 1.3 | #7a4f2c |
| H3 | 18px | 700 | 1.4 | #7a4f2c |
| Body | 14px | 400 | 1.5 | #5d4037 |
| Button | 14px | 700 | 1 | #fff / #7a4f2c |

## Для импорта в Figma

### Text Styles названия
- `Heading/H1` — 28px, Bold
- `Heading/H2` — 24px, Bold
- `Heading/H3` — 18px, Bold
- `Body/Regular` — 14px, Regular
- `Body/Button` — 14px, Bold
- `Body/Label` — 14px, Bold
- `Body/Badge` — 14px, Bold
- `Body/Error` — 14px, Bold
- `Body/Small` — 12px, Regular

## Примеры использования

### Заголовки
```
H1: Ферма (главный заголовок страницы)
H2: Профиль, Магазин, Грядка... (заголовки секций)
H3: Фруктовый салат, Овощной салат... (подзаголовки)
```

### Кнопки
```
[Войти] [Купить] [Продать] [Посадить] [Собрать]
```

### Лейблы форм
```
Email
Пароль
Логин
```

### Бейджи
```
[Уровень: 5] [Баланс: 100 ₽] [Салатов съедено: 2]
```
